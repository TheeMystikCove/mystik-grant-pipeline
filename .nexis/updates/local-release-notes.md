# ✦ What's New in Grant Engine
*(Operator & Researcher Release Notes)*

---

## NEXIS Admin Copilot
*v1.0.0 — March 2026*

NEXIS is now available in Grant Engine as an operational intelligence copilot. It assists with:
- Opportunity scoring and prioritization support
- Grant research queries (powered by Perplexity for live data)
- Pipeline intelligence and status tracking
- Analysis and recommendation generation

**Mode available:** Admin Copilot only

---

## Identity Integration (MYSK-ID)
*v1.0.0 — March 2026*

MYSK-ID is enabled. Grant overlay tracks active opportunities, scoring history, and eligibility status per member. Grant Engine operates at the most restricted data access tier — NEXIS has access only to what is necessary for grant pipeline operations.

---

## Update Intelligence Layer
*v1.1.0 — March 2026*

Grant Engine is now connected to nx.CHRONICLE. Changes that affect operator experience will be documented here.
